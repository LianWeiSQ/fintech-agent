# ClawHub Picks

- `verify-claims` (`installed`): multi-organization verification
- `fact-check` (`selected`): single-claim verdict
- `fact-checker` (`selected`): draft-level report audit

# Audit Focus

- numbers: price, move, size, date, tenor
- attribution: policy to asset, macro to rates, sentiment to sector
- quotes: official statements and media wording

# Guardrails

- validate first, then downgrade wording
- preserve `degraded_reason` for uncertain statements
- every audit result must point back to evidence
